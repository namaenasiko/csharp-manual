# switch文
当面はif文でよい。詳しくは以下のサイトを参照のこと。
http://ufcpp.net/study/csharp/st_branch.html#switch

以下、switch文の書き方。

```cs
Switch(変数){
   // 条件1
   case 条件1;
        // 変数が条件1のときの処理
        break;     // C#では必須
   case 条件2; 
        // 変数が条件2のときの処理
        break;
    // : 必要なだけ、caseは書ける
    // これまでの条件のいずれも該当しないとき
    default;
         // その他の処理
         break;
   }
```

## if文との違い
- 条件判断に使うのは、特定の1つの変数
- 条件が多きときに見やすくなる(ifで分岐させるのは、一般的に3つ程度)
- switch文の利用例
 - 状態遷移の分岐(タイトル画面、ゲーム画面、ゲームオーバー画面、クリア画面などを分岐させるとき)
 - HTTPなどのエラーに対して処理を実装するとき
